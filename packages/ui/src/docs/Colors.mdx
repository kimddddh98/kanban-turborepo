# Design Tokens — Colors

**Design Tokens** 컬러 시스템 문서입니다. `@repo/design-tokens` 패키지에서 빌드된 CSS 변수를 기준으로 정의된 팔레트와 시맨틱 컬러를 확인할 수 있습니다.

이 문서는 `packages/design-tokens/build/css/variables.css`에 정의된 컬러 토큰을 설명합니다.  
스토리북 프리뷰에서 design-tokens CSS가 로드되어 있어 모든 `--colors-*`, `--fg-*`, `--bg-*`, `--accent-*` 변수를 사용할 수 있습니다.

## 개요

- **출처**: `@repo/design-tokens` 빌드 결과 (`build/css/variables.css`)
- **팔레트**: Neutral(black/white), Gray, Red, Orange, Yellow, Green, Teal, Blue, Indigo, Purple, Pink (각 100~900 단계)
- **시맨틱 토큰**: 텍스트/배경/강조용 변수 (fg-default, bg-default, accent-default 등)

## CSS 변수 사용법

앱 또는 컴포넌트에서 아래처럼 사용합니다.

```css
.my-element {
  color: var(--fg-default);
  background-color: var(--colors-blue-500);
  border-color: var(--colors-gray-300);
}
```

```tsx
// 인라인 스타일
<div
  style={{
    backgroundColor: 'var(--accent-default)',
    color: 'var(--accent-on-accent)',
  }}
>
  버튼
</div>
```
